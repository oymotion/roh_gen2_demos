# 肌电手环控制 ROHand

## 准备

安装python和pip
进入命令环境，如windows下的command或者linux下的BASH
进入演示项目目录，例如：

```SHELL
cd gForce_ctrled_rohand
```

安装依赖的python库：

```SHELL
pip install -r requirements.txt
```

## 运行

打开`gForce_ctrled_hand.py`并修改设备地址，例如：

```python
NODE_ID = 2
```

运行：

```python
python gForce_ctrled_hand.py
```

按'ctrl-c'退出。
